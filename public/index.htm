Used technologies at Give A Word

1 - AWS EC2

2 - Nginx & SSL configuration

Nginx and ssl are used to configure server flexibly and ensure the security of website.

3 - Code Standard

Use PHP Codesniffer and ESLint to validate the source code.

4 - Swagger

Swagger allows members to describe the structure of APIs, so that the other members can grasp the purpose of APIs.

5 - Bootstrap Responsive

6 - CRUD Words

Create, edit, delete items. After login, user can give their words, or take words that inspire them.

7 - Redis Cache

Redis is used as a database, cache and message broker.

8 - DynamoDB

DynamoDB is a fast and flexible NoSQL database service for all applications that need consistent, single-digit-millisecond latency at any scale.

9 - Fulltext search

Apply fulltext index to search any keywords in message.

10 - Elastic search

Elasticsearch is a document-oriented database consisting of clusters, indexes and documents. It's used for a fast full-text search and data analysis.
* Because of EC2 memory limitation, I can't run elastic docker on EC2. I can show you at my local.

11 - Send email

In case, user forgot their password, and they'd like to reset. System will send them an email with the link to reset password.

12 - RabbitMQ Help Chat

RabbitMQ is a message broker that provides ways to create queues for connected applications to exchange messages. In this website, RabbitMQ is used to build a help chat.























13 - Canvas & Python

PHP and Python work together
